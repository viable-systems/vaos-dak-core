# @vaos/dak-core

Deterministic autonomy-kernel primitives used by VAOS.

## Scope

- autonomy event and stream types
- repository contracts and implementations
- ledger, lease, reducer, tick engine
- introspection, memory retrieval, adaptation, retry policy

## Install

```bash
npm install @vaos/dak-core
```

## Build

```bash
npm run build
```
